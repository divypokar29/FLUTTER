Looping statements ::: 

there are 3 types of loop 

1) for loop 
2) while loop 
3) do..while loop 


1) for loop ::: for loop is a sequence controlled loop 

        which contain 3 counters 

        inintlization 
        condition 
        updation. 

    syntax : 

        for(initlization;condition;updation)
        {
            statement;
        }


2) while loop : entry controlled loop 

    it check condition at entry level and if condition goes false it will 
    terminate loop. 

    syntax : 

    inilization;
    while (condition)
    {
        updation;
        statement;
    }

3) Do..while loop : 

    syntax : 

    do 
    {
        statement;
        updation;
    }while (condition);

    do while loop check condition at exit level.
