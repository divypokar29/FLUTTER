

<1>. Benefits of Using Flutter Over Other Cross-Platform Frameworks

    => Flutter is a popular UI toolkit developed by Google for building mobile,
       web, and desktop applications using a single codebase.
    
       1). Single Codebase
       2). Fast Development with Hot Reload
       3). Beautiful User Interface
       4). High Performance
       5). Open Source and Free
       6). Supports Multiple Platforms





<2>. Role of Dart in Flutter and Its Advantages

    =>Dart is the programming language used to develop Flutter applications.

     Advantages.
     -----------
     1). Easy to Learn
     2). Fast Performance
     3). Hot Reload Support
     4). Object-Oriented Language
     5). Strong Typing System
     6). Rich Standard Library
     7). Asynchronous Programming





<3>. Steps to Set Up a Flutter Development Environment

    => Step 1: Install Flutter SDK.
       Step 2: Set Environment Path.
       Step 3: Install Android Studio .  
       Step 4: Install Flutter and Dart Plugins.
       Step 5: Check Installation.
             - flutter doctor.
       Step 6: Create a Flutter Project.
             - flutter create myapp.
       Step 7: Run the Application.
             - flutter run.





<4>. Basic Structure of a Flutter App

    1. main.dart
      - main.dart is the entry point of every Flutter application.

    2. Main Function
      - The main() function is the starting point of the app.

    3. runApp()
      - runApp() attaches the widget to the screen.

    4. Widget Tree
      - Flutter UI is built using widgets.
        Widgets are arranged in a hierarchical structure called the Widget Tree.
