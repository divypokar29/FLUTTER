<1>. Explain the fundamental data types in Dart (int, double, String, List, Map, etc.) and
     their uses
 
    => 1). int:

           The int data type stores whole numbers.

           Uses:
               - Counting values
               - Age
               - Roll numbers
               - Integer calculations

        2). double

            The double data type stores decimal numbers.

            Uses :
                 - Currency values
                 - Measurements
                 - Percentage calculations

        3). String

           The String data type stores text.

           Uses:
               - Names
               - Messages
               - User input
               - Text display

        4). bool

            The bool data type stores true or false values.

            Uses:
                - Conditions
                - Login status
                - Validation checks

        5). List

            A List stores multiple values in a single variable.

            Uses:
                - Store collections of items
                - Product lists
                - Student names
                - Dynamic data

        6). Map

            A Map stores data in key-value pairs.

            Uses:
                - JSON data
                - User profiles
                - Database records

        7). dynamic

            The dynamic keyword allows storing any type of value.

            Uses:
                - Flexible data handling
                - API responses


<2>. Describe control structures in Dart with examples of if, else, for,
     while, and switch.

     =>1). if Statement:

           Used to execute code when a condition is true.

           Example:
                  int age = 18;
                  
                  if(age >= 18){
                    print("Eligible to vote");
                  }
        2). if-else Statement:

            Used when there are two possible conditions.

            Example:
                   int marks = 40;
                   
                   if(marks >= 35){
                     print("Pass");
                   }else{
                     print("Fail");
                   }
        3). for Loop:

            Used to repeat code multiple times.
            
            Example:
                   for(int i = 1; i <= 5; i++){
                     print(i);
                   }
        4). while Loop:

            Executes code while the condition is true.
            
            Example:
                    int i = 1;
                    
                    while(i <= 5){
                      print(i);
                      i++;
                    }
        5). switch Statement:

            Used to select one option from multiple choices.
            
            Example:
                    int day = 2;
                    
                    switch(day){
                      case 1:
                        print("Monday");
                        break;
                    
                      case 2:
                        print("Tuesday");
                        break;
                    
                      default:
                        print("Invalid Day");
                    }
