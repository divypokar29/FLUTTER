 widget in flutter 

 -------------------------
 widget are the core vuilding block of a Flutter application
 Everything you see on the screem ,from layout structure to individual butto, is a widget
 Flutter widger system is designed to be flexible and powerfulle , allowing for the creation of highly customized

 ---------------------------------------------------
 1> Types of widget
 widget can be classified into two main categories:

   Stateless widget :
       Do not maintain state . once built ,their properties can't be changed
       
       Example : A simple text lable, icon , or button that does not change upon user interaction


   Stateful widget : 
       Maintain state and can change over time , their appearance can change in response to user input or other 
       events.

======================================================================================================
A Container is a versatile widget in flutter that can hold other widgets.

It acts like a box that can contain a single chiled widget and can be customized in mainy ways .

key properties of Container

child : The widget inside the container . it can be any widget (e.g., text , image , Column ,Row , etc.),

Padding : add space inside the container around the child widget.

margin : adds outside the container , separating it from other widgets.

color : sets the background color of the container.

width and height : sets the width and height of the container.

borderRadius : rounds the corners of the container.

BoxDecoration : used to apply more advanced styling to the container , such as borders , gradients , and shadows.

alignment : Controll the position of the child widget inside the container.

constraints : Used to set constraints like minimum or maximum width and height for the container.

decoration : allows you to apply styling like borders , gradients , and shadows to the container.


e.g. Container (
  constraints: BoxConstraints(
    minHeight: 100,
    maxWidth: 200,
    maxHeight: 300,
    minWidth: 150,
    child: text("with constraints"),
  )
)
